◆ 修飾詞modifiers

Java的修飾詞由存取權限修飾詞及非存取權限修飾詞組成

存取權限修飾詞Access Level Modifiers

Java的存取權限修飾詞分為以下四種
修飾詞 同一類別 同一套件 不同套件的子類別 不同套件且非子類別 舉例
public √ √ √ √ public int value=20;
protected √ √ √ protected int value=20;
(default) √ √ int value=20;
private √ private int value=20;

非存取權限修飾詞Non Acess Modifiers

指的是與存取權無關的修飾詞，主要有以下四種
  1. static
    2. 意即「靜態」，用於類別方法或變數。
    private boolean flag = true ;
  2. final
    3. 相當於常數化，宣告後無法更改其宣告的內容，用於類別、方法與變數的實作。
    static final double days = 31 ;
  3. abstract
    4. 用於建立抽象類別及方法實使用。
    protected static final int width = 420 ;
  4. synochronized
    5. 執行緒(threads)專用，代表同時只能被一個執行緒使用。

◆ 回傳值型態

代表使用該方法後，得到結果的資料型態。大部分都是宣告一般資料型態，若使用該方法無回傳值則宣告void(代表空的意思)。

◆ 方法名稱

該方法的名稱。

◆ 參數

即撰寫方法時，方法外部傳進資料的代稱。對於方法而言可有可無。參數的宣告須寫上其資料型態再加上參數名稱。參數名稱是可以隨意亂取，但不能跟全域變數與區域變數的名字相同。


EX : Math.random(); 即為不帶參數的方法。

◆ 區域變數

在該方法內部宣告的變數，其有效範圍(Scope)只限定於該方法內，若在方法外面使用則視為無效。執行完方法後會被系統回收空間。

◆ 程式碼區

程式設計者實作該方法的方式

◆ 回傳值

用該方法欲得到的最終目的。其返回的型態為回傳值型態。若無返回值則不須寫。
